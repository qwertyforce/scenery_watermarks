# scenery_watermark

https://huggingface.co/datasets/qwertyforce/scenery_watermarks  
https://www.kaggle.com/datasets/qwertyforce/scenery-watermarks  
  
~22k images, 512x512, manually annotated from /r/EarthPorn/
